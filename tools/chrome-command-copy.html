    # 🔨 Chrome 工具雕虫小技之 copy

    事情是这样的，有时候我们需要把直接打印在浏览器控制台里的大对象复制出来（有用），这时候我们通过普通的选中右键复制是不行的。
    不要着急，chrome已经为我们想到了这种情况，我们只需要把那个对象右键并选中 Store as global variable，生成一个全局变量，然后我们使用命令copy("刚才生成的那个变量")，就已经复制到你的剪切板了，是不是很简单。
